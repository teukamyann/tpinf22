# Projet API Blog - INF222

## Description
Installation d'une API REST pour gérer des articles de blog.

## Installation
1. `npm install`
2. `node index.js`

## Fonctionnalités
- Affichage des articles (GET)
- Ajout via formulaire (POST)
- Suppression par index (DELETE/GET)
- Recherche par titre
